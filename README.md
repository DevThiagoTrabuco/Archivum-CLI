# Archivum CLI 🗃️

Archivum CLI is a Node.js command-line tool for organizing photo collections into archival PDF documents.

It automatically groups images by filename, applies optional watermarks, and generates one PDF per group, making it suitable for document preservation and historical collections.

## Features

- 📁 Process an entire directory of images.
- 🖼️ Apply image watermarks before PDF generation.
- 📄 Generate one PDF per image group.
- 🗂️ Automatically group images based on filename patterns.
- 🚀 Batch processing for large collections.

## Installation

```bash
git clone https://github.com/DevThiagoTrabuco/Archivum-CLI.git

cd Archivum-CLI

npm install

node archivum.js
```
